üîê 2. Anmeldung

Um auf den Admin-Bereich zugreifen zu koennen, ist eine Anmeldung erforderlich.

So melden Sie sich an

  1. Admin-Seite oeffnen: Navigieren Sie zu /admin/
  2. Zugangsdaten eingeben: Benutzername und Passwort eintragen
  3. Anmelden klicken: Sie werden zum Dashboard weitergeleitet

Benutzerrollen

Rolle Berechtigung Beschreibung
Admin Stufe 3 Vollzugriff auf alle Funktionen inkl. Benutzerverwaltung
Editor Stufe 2 Ergebnisse, News und Forum bearbeiten
Registriert Stufe 1 Forum-Beitraege schreiben
Gast Stufe 0 Nur Lesen

‚öΩ 3. Liga-Editor

Der Liga-Editor ist das Herzstck des Systems. Hier werden alle Spielergebnisse eingetragen und verwaltet.

Ergebnisse eintragen

  1. Liga auswaehlen: Waehlen Sie oben links die gewuenschte Liga aus dem Dropdown
  2. Spieltag waehlen: Navigieren Sie zum entsprechenden Spieltag
  3. Ergebnisse eingeben: Klicken Sie in die Ergebnis-Felder und tragen Sie die Tore ein
  4. Speichern: Klicken Sie auf "Spieltag speichern" oder druecken Sie Strg+S
üí°

Tipp: Mit Tab springen Sie schnell zum naechsten Eingabefeld. Das beschleunigt die Eingabe erheblich!

Spielnotizen hinzufuegen

Fuer jedes Spiel koennen zusaetzliche Informationen hinterlegt werden:

  • Notiz: Kurze Info wie "Verlegt" oder "Abgesagt"
  • Datum/Uhrzeit: Ansto√üzeit aendern
  • Spielbericht: Link zu einem externen Bericht

Klicken Sie auf das Info-Symbol neben einem Spiel, um das Notiz-Fenster zu oeffnen.

Tabelle

Die Tabelle wird automatisch berechnet basierend auf:

  • 3 Punkte fuer einen Sieg
  • 1 Punkt fuer ein Unentschieden
  • 0 Punkte fuer eine Niederlage
  • Sortierung: Punkte ‚Üí Tordifferenz ‚Üí Tore

Tastenkuerzel

üì∞ 4. News & Spielberichte

Verwalten Sie hier alle Spielberichte und Neuigkeiten. News koennen mit Spielen verknuepft werden, sodass sie direkt beim jeweiligen Spiel angezeigt werden.

Neue News erstellen

  1. News-Bereich oeffnen: Gehen Sie zu /admin/news.html
  2. "+ Neue News" klicken: Das Editor-Fenster oeffnet sich
  3. Titel eingeben: Aussagekraeftiger Titel fuer den Bericht
  4. Autor angeben: Name des Verfassers
  5. Kurztext schreiben: Wird als Vorschau angezeigt (2-3 Saetze)
  6. Volltext verfassen: Der komplette Spielbericht
  7. Speichern: Klicken Sie auf "Speichern"

News mit Spielen verknuepfen

Damit ein Spielbericht direkt beim Spiel erscheint:

  1. News bearbeiten: Oeffnen Sie die entsprechende News
  2. Spiel suchen: Im Bereich "Verknuepfte Spiele" Team oder Datum eingeben
  3. Spiel auswaehlen: Klicken Sie auf das gewuenschte Spiel
  4. Speichern: Die Verknuepfung wird gespeichert
‚ÑπÔ∏è

Eine News kann mit mehreren Spielen verknuepft werden (z.B. Saisonrueckblick).

Text formatieren

Die Editor-Toolbar bietet folgende Formatierungen:

Suche und Filter

Nutzen Sie das Suchfeld oben, um nach Titel, Inhalt oder Autor zu suchen. Die Ergebnisse werden paginiert angezeigt (20 pro Seite).

üí¨ 5. Forum-Verwaltung

Das Forum ermoeglicht Diskussionen zwischen Benutzern. Administratoren koennen Kategorien anlegen, Beitraege moderieren und das Forum mit Ligen verknuepfen.

Kategorien verwalten

  1. Forum-Admin oeffnen: Gehen Sie zu /admin/forum.html
  2. Tab "Kategorien" waehlen: Alle Kategorien werden angezeigt
  3. Neue Kategorie: Klicken Sie auf "+ Neue Kategorie"
  4. Name und Beschreibung eingeben
  5. Berechtigungen setzen: Wer darf lesen/schreiben?
  6. Speichern

Berechtigungsstufen

Stufe Wer kann zugreifen?
Oeffentlich (0) Alle, auch Gaeste
Registriert (1) Nur eingeloggte Benutzer
Editor (2) Nur Editoren und Admins
Admin (3) Nur Administratoren

Beitraege moderieren

  • Bearbeiten: Inhalt eines Beitrags aendern
  • Loeschen: Beitrag entfernen (unwiderruflich)
  • Topic sperren: Keine neuen Antworten moeglich
  • Topic anpinnen: Erscheint immer oben in der Liste

Liga-Integration

Im Tab "Liga-Integration" koennen automatisch Spieltags-Topics erstellt werden:

  1. Liga auswaehlen
  2. Liga-Kategorie erstellen: Falls noch nicht vorhanden
  3. Spieltags-Topics erstellen: Bulk-Erstellung fuer alle Spieltage
üí°

Tipp: Die erstellten Topics werden automatisch mit dem jeweiligen Spieltag verknuepft und erscheinen im Liga-Editor.

üìä 7. Punktekorrekturen

Manchmal muessen Punkte korrigiert werden, z.B. bei Lizenzentzug oder Regelverstoessen.

Korrektur eintragen

  1. Dashboard oeffnen
  2. Tab "Punktekorrekturen" waehlen
  3. Liga auswaehlen
  4. "+ Korrektur hinzufuegen" klicken
  5. Team waehlen: Welches Team betroffen ist
  6. Punkte eingeben: Negativ fuer Abzug, positiv fuer Bonus
  7. Grund angeben: Z.B. "Spielereinsatz ohne Spielberechtigung"
  8. Speichern

Beispiele

Situation Punkte
3 Punkte Abzug wegen Regelversto√ü -3
1 Punkt Abzug wegen Ordnungswidrigkeit -1
Spielwertung 3:0 (3 Punkte gutgeschrieben) +3

üîê 8. Benutzerverwaltung

Verwalten Sie hier alle registrierten Benutzer und deren Berechtigungen.

Benutzer bearbeiten

  1. Dashboard oeffnen
  2. Tab "Benutzer" waehlen
  3. Benutzer in der Liste finden
  4. Rolle aendern: Im Dropdown die neue Rolle waehlen

Neuen Benutzer anlegen

Benutzer registrieren sich selbst ueber die Login-Seite. Danach kann ein Admin die Rolle hochstufen.

Passwort zuruecksetzen

Aktuell muss ein neues Passwort direkt in der Datenbank gesetzt werden. Funktion zur Passwort-Reset kommt in einer zukuenftigen Version.

Wichtig: Geben Sie Admin-Rechte nur vertrauenswuerdigen Personen. Admins haben vollen Zugriff auf alle Daten.

üí° 9. Tipps & Tricks

Allgemeine Tipps

  • Browser-Cache leeren: Bei Anzeigeproblemen Strg+F5 druecken
  • Mehrere Tabs: Sie koennen mehrere Admin-Bereiche gleichzeitig offen haben
  • Mobil nutzbar: Das Admin-Panel funktioniert auch auf Tablets

Performance

  • Die API nutzt Caching - Aenderungen koennen bis zu 5 Minuten brauchen, bis sie oeffentlich sichtbar sind
  • Bei vielen gleichzeitigen Aenderungen: Zwischendurch speichern

Fehlerbehebung

Problem Loesung
Seite laedt nicht Browser-Cache leeren, Seite neu laden
Aenderungen werden nicht gespeichert Session abgelaufen - neu einloggen
"Keine Berechtigung" Fehler Benutzerrolle pruefen, ggf. Admin kontaktieren
Tabelle zeigt falsche Punkte Punktekorrekturen pruefen
News erscheint nicht beim Spiel Spielverknuepfung in der News pruefen

Backup

Die Datenbank liegt unter /data/lmo.sqlite. Regelmaessige Backups dieser Datei werden empfohlen.

üí°

Profi-Tipp: Legen Sie vor groesseren Aenderungen immer ein Backup der SQLite-Datenbank an!
